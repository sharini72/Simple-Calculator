# SIMPLE CALCULATOR 🧮

A simple and interactive calculator application built using **HTML, CSS, and JavaScript**.  
This project is designed to perform basic arithmetic operations with a clean and user-friendly interface.

## Features 🚀

- Perform basic mathematical operations like:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)

- Interactive calculator layout with buttons and a display screen.
- Handles user input and displays results instantly.
- Includes a clear button to reset calculations.
- Provides real-time calculation output.
- Supports keyboard input for a better user experience.
- Styled with CSS to create a modern and responsive design.

## Technologies Used 💻

- HTML5 - Structure of the calculator
- CSS3 - Styling and layout design
- JavaScript - Functionality and calculations

## Project Description 📌

This calculator project demonstrates the use of front-end web development concepts by combining HTML, CSS, and JavaScript. It focuses on creating a functional interface, handling user actions, and providing accurate calculations in real time.

## How to Use ▶️

1. Open the calculator application.
2. Enter numbers using buttons or your keyboard.
3. Choose the required arithmetic operation.
4. View the result instantly on the display screen.
